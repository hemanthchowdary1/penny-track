# 💸 PennyTrack – Smart Expense Tracker

A full-stack expense tracking application built using React and Django REST Framework.

## 🚀 Features
- Add / delete expenses
- Budget tracking
- AI insights
- Charts & analytics
- JWT authentication
- CSV export
- Responsive UI

## 🛠 Tech Stack
- Frontend: React
- Backend: Django REST Framework
- Auth: JWT
- Charts: Chart.js

## ⚠️ Note
Environment variables and API keys are not included for security reasons.
